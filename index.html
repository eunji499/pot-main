About me

사용자 경험을 기반으로 직관적이고 트렌디한 웹 디자인을 구현합니다.
데이터와 흐름을 고려한 설계를 통해, 사용자가 머무르고 싶은 화면을 만듭니다.
디자인과 사용성, 두 가지를 모두 만족시키는 결과물을 만들기 위하여 노력합니다.

Design Strengths

  • 시각적 완성도를 높이는 UI 디자인

    디자인 요소 간의 균형과 디테일을 통해 완성도 높은 UI를 구현합니다.단순한 미적 표현을 넘어, 브랜드 가치를 전달하는 디자인을 만듭니다.

  • 사용자 중심으로 사고하는 디자인 접근

    사용자의 행동과 흐름을 기반으로 직관적인 인터페이스를 설계합니다.누구나 쉽게 이해하고 사용할 수 있는 경험을 만드는 데 집중합니다.

  • 작은 디테일까지 설계하는 완성도

    보이지 않는 부분까지 고민하는 디테일로 디자인의 완성도를 높입니다.작은 차이가 전체 경험을 바꾼다고 믿습니다.

Collaboration Skills

  • 원활한 커뮤니케이션을 위한 노력

    기획자와 개발자의 의견을 이해하고자 적극적으로 소통하며, 작업 의도를 정확하게 전달하려고 노력합니다. 협업 과정에서 생기는 차이를 줄이기 위해 꾸준히 커뮤니케이션합니다.

  • 협업을 고려한 디자인 이해

    디자인뿐 아니라 개발 과정에 대한 이해를 바탕으로, 구현 가능성을 고려한 작업을 지향합니다.협업의 흐름을 이해하며 효율적인 결과물을 만들고자 합니다.

  • 피드백을 통한 성장

    피드백을 긍정적으로 받아들이고, 이를 빠르게 반영하여 더 나은 결과물을 만들기 위해 노력합니다.협업 과정 속에서 배우고 성장하는 디자이너입니다.
